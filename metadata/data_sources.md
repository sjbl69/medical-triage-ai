# Data Sources

This project uses a **synthetic dataset** generated programmatically.

## Source

- Generated using rule-based medical triage logic
- No real patient data used
- Designed for prototyping only

## Content

- Symptoms (chest pain, bleeding, fever, etc.)
- Associated triage responses:
  - URGENT
  - NON-URGENT

## Purpose

- Train a language model for medical triage classification
- Evaluate safety and response consistency

## Limitations

- Not clinically validated
- Not suitable for real-world medical usage
